# Quick Start Guide for Educators 🎓

## Welcome to QuizAI!

This guide will help you get started with creating and managing quizzes in minutes.

## Step 1: Sign Up (2 minutes)

1. Go to the website
2. Click **"Sign In"** in the top right
3. Choose:
   - **Google Sign-In** (fastest) OR
   - **Email & Password** (click "Sign up" tab)
4. You're in! 🎉

## Step 2: Set Up Your AI Provider (3 minutes)

1. Click **"Settings"** in the navbar
2. Scroll to **"API Keys"** section
3. **Recommended:** Get a free Gemini API key
   - Click the link next to "Google Gemini"
   - Sign in to Google AI Studio
   - Click "Create API key"
   - Copy the key (starts with `AIza...`)
4. Paste key in the field
5. Click **"Test"** button ✅
6. If it says "✓ API key is valid" → Click **"Save All API Keys"**

> **Tip:** Gemini has a generous free tier, perfect for getting started!

## Step 3: Create Your First Quiz (1 minute)

1. Click **"Generate"** in the navbar
2. Enter a quiz title (e.g., "Introduction to React")
3. Paste your content in the text area (200-1000 words works best)
4. Choose:
   - Number of questions (try 5 to start)
   - Question type (try "Mixed" for variety)
5. Click **"Generate Quiz"**
6. Wait 5-10 seconds... ⏳

## Step 4: Review Your Quiz (1 minute)

You'll see a success page! 🎉

**You can:**
- ✅ Copy the quiz link
- ✅ Click **"View Quiz"** → Manage questions
- ✅ Click **"Preview"** → Test it yourself
- ✅ Click **"Dashboard"** → See all your quizzes

**Recommended:** Click **"Preview"** to test the quiz yourself first!

## Step 5: Share with Students (30 seconds)

Two ways to share:

**Option 1: From Success Page**
1. Click **"Copy"** button next to the link
2. Paste in email/LMS/chat
3. Done! ✅

**Option 2: From Dashboard**
1. Click **"Dashboard"**
2. Find your quiz
3. Click the **Share** button (green icon)
4. Link copied! ✅

## Step 6: Manage Your Quiz (Anytime)

### Edit Questions

1. Go to **Dashboard**
2. Click **"Manage"** (eye icon) on your quiz
3. Scroll to questions
4. For each question:
   - Click **"Edit"** → Modify manually
   - Click **"Regenerate"** → AI creates new question
5. Changes save automatically!

### View Analytics

In the manage page, you'll see:
- 📊 Total questions
- 👥 How many students attempted
- 📈 Average score
- 🔄 Quiz status (enabled/disabled)

### Export Results

1. In manage page, scroll to "Recent Attempts"
2. Click **"Export CSV"**
3. Open in Excel or Google Sheets
4. Import to your gradebook!

### Disable Quiz After Deadline

1. In manage page, click **"Enabled"** toggle
2. It changes to **"Disabled"**
3. Students now see "Quiz not available"
4. Re-enable anytime by clicking toggle again

## Tips for Success

### Creating Better Quizzes

✅ **DO:**
- Use clear, well-written content
- Include key concepts and facts
- Start with 5-10 questions
- Preview before sharing
- Test quiz yourself first

❌ **DON'T:**
- Use very complex or technical text
- Generate 20+ questions at once
- Skip previewing
- Share without testing

### Question Types

**Multiple Choice (4 options):**
- Best for detailed knowledge
- Tests understanding
- Slightly harder

**True/False (2 options):**
- Best for facts
- Quick assessment
- Easy to answer

**Mixed (Recommended):**
- Variety of both types
- More engaging
- Better assessment

### Managing Quizzes

✅ **Best Practices:**
- Preview all quizzes
- Edit unclear questions
- Disable after deadline
- Export results regularly
- Keep organized with clear titles

## Troubleshooting

### "API key test failed"
1. Check you copied the complete key
2. Verify key is from correct provider
3. Check account has credits
4. Try different provider (Gemini or Groq)

### "Quiz generation failed"
1. Make sure API key is saved
2. Test API key first
3. Check content isn't too short/long
4. Try different AI provider

### "No quizzes showing in Dashboard"
1. Make sure you created a quiz
2. Check you're signed in
3. Refresh page
4. Clear browser cache

### "Can't export results"
1. Make sure quiz has attempts
2. Check browser allows downloads
3. Try different browser
